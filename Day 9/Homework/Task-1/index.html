What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML elements are the building blocks of HTML pages and are represented by tags like <head>, <body>, <h1>, etc.

What is a Semantic Tag?

Semantic tags like <article>, <section>, <header>, and <footer> convey meaning and structure to your content. They help browsers, search engines, and assistive technologies better understand the purpose of different parts of your webpage, improving accessibility and SEO.

Difference Between id and class

In HTML and CSS:

  • id: Must be unique on a page, can only be used once. Referenced with # in CSS.
  • class: Can be reused multiple times on a page. Referenced with . in CSS.
  • An element can have multiple classes but only one id.
  • ids have higher specificity than classes in CSS.

What is a <summary> Element?

The <summary> element is used as a visible heading for its parent <details> element. It shows a disclosure widget that the user can click to show/hide the content. It's particularly useful for creating expandable/collapsible sections like FAQs, accordions, or info sections as demonstrated in this handbook.

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of HTML documents. It controls layout, colors, fonts, and responsive design. CSS can be applied in three ways:

  • Inline: Using the style attribute directly in HTML elements
  • Internal: Using the <style> tag in the document head
  • External: Linking to an external .css file with <link> tag
